SilimaSoftware

Who?

Founded in 2021, SilimaSoftware is a small South African start-up focused on building meaningful software products and services. Being a small team of local developers primarily working on core service offerings, we occasionally outsource freelancer developers for larger projects.

Why?

Our grand plan is to design and create an end-to-end general solution that encompasses all elements of DevOps and the Software development lifecycle into one glorious, automated hosting platform. One platform to rule them all. In addition to building our products, we offer our development services to you.

What?

You might be thinking, what can we do for you? Implementing Agile DevOps methodologies allows us to keep you, the client, in the loop and work closely at every phase of the development lifecycle. Additionally, continue to improve and grow solutions with the ever-evolving business requirements.

How?

Software Development

Python is our choice of language, using frameworks such as Django, Flask, and FastAPI to build secure scalable websites, web applications and APIs. The unmistakable choices for Web UI are JavaScript or jQuery. Perhaps React depending on the size, scale, and type of project.

Database

Postgresql the RDBMS that could. Furthermore considering the benefits of using Postgresql with Djangos built-in object-relational mapping it was a no-brainer.

CI/CD

Distributed Jenkins build nodes running in containers building containers. Using pipeline scripts to define builds, testing, and deployments.

Containers

Containerize all the things. We love Containers. Using tools like Docker, Traefik, Nginx, and others to manage containers in production securely has never been this simple. We have deployed our containers on a range of virtual machine vendors. If it runs Linux chances are it will work.
