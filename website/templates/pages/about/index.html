How the magic happens

Our technology stack

Software Development

Python is our choice of language, using frameworks such as Django, Flask, and FastAPI to build secure scalable websites, web applications and APIs. The unmistakable choices for Web UI are JavaScript or jQuery. Perhaps React depending on the size, scale, and type of project.

Database

Postgresql the RDBMS that could. Furthermore considering all the benefits of using Postgresql with Djangos built-in object-relational mapping, it was a no-brainer.

CI/CD

Distributed Jenkins build nodes running in containers, building containers. Using pipeline scripts to define builds, testing, and deployments.

Containers

Containerize all the things. We love Containers. Using tools like Docker, Traefik, Nginx, and others to manage containers in production securely has never been this simple. We have deployed our containers on a range of virtual machine vendors. If it runs Linux, chances are it will work.
